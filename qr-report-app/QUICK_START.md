# 🚀 QUICK START GUIDE

## For First-Time Users

### Option 1: Double-Click Method (Easiest)

1. **Double-click** `START_APP.bat`
2. Wait for the browser to open
3. Start using the app!

### Option 2: Command Line Method

1. **Open Command Prompt** (Press Win+R, type `cmd`, press Enter)

2. **Navigate to project folder:**
   ```
   cd "d:\DEVELOPMENT REPORTS\QR-REPORT\qr-report-app"
   ```

3. **Install dependencies** (first time only):
   ```
   npm install
   ```

4. **Start the app:**
   ```
   npm run dev
   ```

5. **Open browser** to: `http://localhost:5173`

---

## Using the App

### 1. Upload Your Data
- Click **"Upload CSV"** button
- Select your collection data CSV file
- Data will load automatically

### 2. View Reports
- **Table View**: See detailed ward-wise data
- **Charts View**: Visual representations

### 3. Export Reports
- **PDF**: Click "Export PDF" for styled report
- **Excel**: Click "Export Excel" for spreadsheet

---

## Stopping the App

- In the command window, press **Ctrl+C**
- Or simply close the command window
